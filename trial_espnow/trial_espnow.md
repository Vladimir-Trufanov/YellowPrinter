### [Как узнать MAC-адрес](https://arduinokitproject.com/esp32-mac-address-tutorial/)

MAC-адрес (Media Access Control) — это уникальный идентификатор сетевого устройства, состоящий из 12 шестнадцатеричных символов, сгруппированных в пары, например AA:BB:CC:DD:EE:FF.

#### [Скетч "Узнать mac-адрес контроллера"](GetMac/GetMac.ino)

Каждое устройство, подключаемое к сети, оснащено контроллером сетевого интерфейса (Network Interface Controller, NIC), который содержит схемы, необходимые для связи по сетевым протоколам, таким как Wi-Fi, Bluetooth или Ethernet. Во время производства производители присваивают каждому контроллеру сетевого интерфейса уникальный MAC-адрес, который навсегда сохраняется в аппаратной части устройства.

MAC-адреса необходимы для отправки и получения данных по сети. IP-адреса идентифицируют сетевое подключение, а MAC-адреса — аппаратное обеспечение. В отличие от IP-адресов, которые часто меняются, MAC-адреса статичны и используются только в локальной сети.

Хотя MAC-адрес обычно встроен в аппаратную часть устройства, многие системы, в том числе ESP32, позволяют изменять его программно. Однако этот пользовательский MAC-адрес является временным и после сброса настроек чипа возвращается к значению по умолчанию, если только вы не запрограммируете ESP32 на его установку при каждой загрузке.

ESP32 имеет несколько MAC-адресов, каждый из которых соответствует отдельному сетевому интерфейсу:

```
Станция Wi-Fi       (STA)
Точка доступа Wi-Fi (AP)
Интерфейс Bluetooth (классический и BLE)
Ethernet
```


Базовый MAC-адрес, хранящийся в eFuse или во внешнем хранилище, служит основой для генерации MAC-адресов для этих интерфейсов.

- Базовая MAC-адресация используется станцией Wi-Fi (STA);
- MAC-адрес точки доступа Wi-Fi (AP) генерируется путем увеличения последнего октета базового MAC-адреса на единицу.
- MAC-адрес Bluetooth получается путем добавления двух к последнему октету базового MAC-адреса.
- MAC-адрес Ethernet создается путем добавления трех к последнему октету базового MAC-адреса.

```
Интерфейс     MAC-адрес	                        Пример
------------- -------------------------------   -----------------
Станция Wi-Fi base_mac       	                80:7D:3A:CB:12:5C
Wi-Fi SoftAP  base_mac +1 к последнему октету	80:7D:3A:CB:12:5D
Bluetooth	  base_mac +2 к последнему октету	80:7D:3A:CB:12:5E
Ethernet	  base_mac +3 к последнему октету	80:7D:3A:CB:12:5F
```
